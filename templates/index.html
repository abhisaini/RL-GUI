Grid World

Instructions

  • You are recommended to use computer because the grid world may hamper in mobile screen
  • This is an instance of a grid world where every grid element has a value associated with it in which an agent has to get more one to other end by taking four actions: left, right, up and down.(can be made using arrow keys)
  • The value of a grid element is represented in the form of Colors. Darker the color of the grid element the more valuable it is.
  • So, the problem is to reach the terminal point maximising the cumulative gain in minimum number of steps.
  • The following experiment is to know the difference between the optimal solution that there is and intuitive solutions found by humans.
  • If you visit a cell second time it will not count to total reward.
  • Please have a look at demo.
  • Once you reach the red end, data will be submitted automatically. You can exit then.
